# 💼 HustleLedger

HustleLedger is a financial record-keeping application designed to help informal businesses build a trusted financial track record.

## Features

- User Registration
- Secure Login
- Record Sales
- Record Expenses
- Dashboard Analytics
- Excel Export
- PDF Reports
- SQLite Database
- Streamlit Cloud Ready

## Installation

```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```
